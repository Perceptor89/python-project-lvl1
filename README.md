### About Brain games
With this app you can play five different games. Train your brain and have fun!

### Minimum requirements
To use this app you need:

| Program         | Version        |
|-----------------|----------------|
| python          | ^3.8           |
| prompt          | ^0.4.1         |

### Installation
For installation use command:
```
make quick-install
```
### Start games with commands:
```
brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
```
